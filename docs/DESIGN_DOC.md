1. DESIGN VISION
----------------
GigNest is a modern SaaS-style freelance marketplace.
The design emphasizes:
• Trust
• Clarity
• Simplicity
• Skill-based discovery
• Smooth micro-interactions

The UI should feel like a startup product, not a college project.

2. DESIGN PRINCIPLES
--------------------
• Minimalist layout
• Generous white space
• Card-based components
• Clear hierarchy
• Responsive-first design
• Subtle, smooth animations

3. COLOR SYSTEM
---------------
Primary: Indigo (#4F46E5)
Accent: Cyan (#06B6D4)
Background Light: #F9FAFB
Text Dark: #111827
Muted Gray: #6B7280

Optional Dark Mode:
Background: #111827
Card: #1F2937

4. TYPOGRAPHY
-------------
Headings: Poppins / Montserrat
Body: Inter
Buttons: Medium weight

5. LAYOUT SYSTEM
----------------
• 12-column grid
• Max width container: 1200px
• 16–24px padding
• 12px border radius
• Soft shadow (0 4px 12px rgba(0,0,0,0.08))

6. GLOBAL ANIMATIONS
--------------------
• Page load fade-in (250ms)
• Card hover lift (translateY -4px)
• Button scale (1.03 on hover)
• Sidebar slide transition (300ms)
• Toast notification slide-right

7. PUBLIC PAGES

Landing Page

Sections:
• Hero with search bar
